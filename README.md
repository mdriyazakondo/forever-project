## 🚀 Quick Start

### Prerequisites

- Node.js (version 16 or higher recommended)
- npm (or yarn/pnpm)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mdriyazakondo/forever-project.git
   cd forever-project
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Visit `http://localhost:5173`
